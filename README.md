## Installation

### Requirements

#### Python, packages

- Python (3.6)

```
pip install -r requirements.txt
```

#### Secrets
`.secrets/base.json`

```json
{
  "SECRET_KEY": "<Django SECRET_KEY>",
  "FACEBOOK_APP_ID": "<Facebook APP ID>",
  "FACEBOOK_APP_SECRET": "<Facebook APP Secrets>"
}
```
